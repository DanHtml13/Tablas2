Tema 1: Texto

Propiedad Valor Inicial Descripción
color black Define el color del texto. Puede especificarse con nombres de colores (como rojo), códigos hexadecimales (#ff0000), valores RGB (rgb(255, 0, 0)), RGBA, HSL, etc.
font-family serif Especifica la fuente del texto. Puede ser una lista de fuentes o una familia genérica (como serif, sans-serif, monospace, etc.).
font-size medium Define el tamaño de la fuente. Se puede especificar en unidades relativas (em, rem) o absolutas (px, pt), entre otras.
font-weight normal Controla el grosor de las letras. Los valores comunes son normal, bold (negrita), lighter, o valores numéricos (100-900).
font-style normal Especifica el estilo de la fuente. Los valores pueden ser normal, italic (cursiva) u oblique (un tipo de cursiva ligera).
font-variant normal Controla las variaciones tipográficas de la fuente, como normal o small-caps (letras en minúsculas transformadas a minúsculas pequeñas).
font normal Propiedad abreviada para definir todos los aspectos de la fuente (familia, tamaño, estilo, grosor, etc.).
text-align left Alinea el texto dentro de su contenedor. Los valores comunes son left, right, center, y justify.
line-height normal Define la altura de las líneas de texto, controlando el espacio entre las líneas. Se puede establecer como un número, longitud o porcentaje.
text-decoration none Define la decoración del texto, como underline (subrayado), line-through (tachado), overline (sobre línea), o none (sin decoración).
text-transform none Controla la transformación del texto en términos de mayúsculas y minúsculas. Los valores incluyen uppercase, lowercase, capitalize, y none.
vertical-align baseline Controla la alineación vertical de los elementos en línea. Los valores comunes son baseline (alineación a la línea base), middle, top y bottom.
text-indent 0 Establece la sangría de la primera línea de un bloque de texto (por ejemplo, en un párrafo). Puede definirse en píxeles, ems, etc.
letter-spacing normal Controla el espaciado entre las letras. El valor por defecto es normal, pero puede modificarse con unidades de longitud como px, em, etc.
word-spacing normal Controla el espacio entre palabras. El valor por defecto es normal, pero también puede definirse en unidades como px, em, etc.
white-space normal Define cómo se maneja el espacio en blanco dentro de un elemento. Los valores incluyen normal (colapsar el espacio en blanco), nowrap (evitar saltos de línea), pre (preservar los espacios), etc.

Tema 2: Enlaces

Característica Propiedades Descripción
Tamaño, Color y Decoración color Establece el color del texto del enlace. Se pueden utilizar valores de color CSS como nombres (por ejemplo, blue), valores hexadecimales (#ff6347), o valores RGB (rgb(255, 0, 0)).
text-decoration Controla las decoraciones del enlace, como subrayado (underline), tachado (line-through), o sobre línea (overline). El valor por defecto es none, es decir, sin decoración.
text-transform Modifica la forma en que se presenta el texto, como convertirlo en mayúsculas (uppercase), minúsculas (lowercase), o capitalizar la primera letra de cada palabra (capitalize).
Decoración Personalizada border-bottom Añade un borde en la parte inferior del enlace, útil para simular un subrayado personalizado con grosor y color definidos.
text-shadow Aplica una sombra al texto del enlace. Puedes especificar el desplazamiento, el difuminado y el color de la sombra. Ejemplo: text-shadow: 2px 2px 4px rgba(0, 0, 0, 0.3);.
outline Similar a un borde, pero no afecta el flujo del diseño. Se usa generalmente para resaltar un enlace cuando recibe el foco (por ejemplo, al hacer clic o al navegar con el teclado).

Tema 3: Imágenes

Característica Propiedades Descripción
Anchura y Altura Width Define el ancho de la imagen. Se puede especificar en unidades como px, %, em, vw (anchura de la ventana) u otras unidades relativas.
Height Establece la altura de la imagen. Al igual que la propiedad width, puede ser definida en unidades como px, %, em, vh (altura de la ventana), entre otras.
